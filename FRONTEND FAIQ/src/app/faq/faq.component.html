How to Add Student Information?

Users can press the button on the navbar that is available on the page. The user needs to select "Add Student" if the page is clicked, it will change to the student page and the user can enter the student's information.


Can I correct the student information I have made?

Of course, the User can change the information that has been saved. Users only need to go to "Student List" in the navbar and select the student information they want to create. after selecting the user can press the "update" button and will be taken to the next page, which is the "updatestudent" page. The user can change the information and press the "Confirmupdate" button


Can I delete the student information I have made?

Of course, the User can delete the information that has been saved. Just like Update students, users only need to go to "Student List" in the navbar and select the student information they want to create. after selecting the user can press the "delete" button and the information will be removed from the system and database
